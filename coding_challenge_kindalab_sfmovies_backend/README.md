# Estructura

El JSON que devuelve la API de SFMOVIES tiene la siguiente estructura:
```JSON
[
    {
        "title": "string",
        "release_year": "string",
        "locations": "string",
        "production_company": "string",
        "distributor": "string",
        "director": "string",
        "writer": "string",
        "actor_1": "string",
        "actor_2": "string",
        "actor_3": "string"
    }
]
````
